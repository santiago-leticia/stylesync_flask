<h1>Ideia do projeto:</h1>

O objetivo principal do projeto do curso de Flask: desenvolvendo APIs e aplicações web com MongoDB é desenvolver uma aplicação web para a "Style Sync" de forma rápida e eficiente.

Inicialmente, a aplicação exibirá informações de produtos (nome e valor) em uma página web. A página será estática, sem conteúdo dinâmico vindo de uma API ou banco de dados.

<h2>O que foi ensinado:</h2>

- Criar um servidor web utilizando a biblioteca padrão do Python (WSGI) para servir a aplicação.
- Separar as responsabilidades entre o back-end (lógica do servidor em Python) e o front-end (estrutura e estilo da página web em HTML).
- Integrar o HTML com o Python para exibir informações dinâmicas de produtos na página web.
- Simular o consumo de uma API ou banco de dados para obter os dados dos produtos.
- Iterar sobre os produtos e atualizar o HTML dinamicamente com as informações.
- No entanto, o curso não se limita a isso. Ao final desta aula, é levantada a questão da escalabilidade da aplicação para múltiplas rotas (páginas de contato, sobre, FAQ, etc.). A solução para esse problema será abordada nos próximos vídeos, com a introdução de frameworks que facilitam o desenvolvimento de aplicações web mais complexas.
